# Setup Guide

This project requires both Frontend and Backend to be running.

## 1. Quick Global Setup

**Windows (PowerShell)**:
```powershell
# In one terminal:
cd backend; npm install; npm run dev

# In another terminal:
cd frontend; npm install; npm run dev
```
