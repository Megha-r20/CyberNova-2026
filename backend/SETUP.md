# Backend Setup Guide

1. **Navigate to backend**:
   ```bash
   cd backend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   - Copy `.env.example` to `.env`.
   - Update `ADMIN_KEY` if desired.

4. **Start Server**:
   ```bash
   npm run dev
   ```

5. **Verify**:
   - Console should show a box with "Status: Online".
   - `data/cybernova_registrations.xlsx` will be auto-created.
